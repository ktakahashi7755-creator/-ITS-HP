1.1 技術スタック

レイヤー技術役割特記
マークアップHTML5構造・意味論・アクセシビリティセマンティック要素必須（section/article/header/footer/main）
スタイリングCSS3 + Custom Propertiesデザイントークン・レイアウト・アニメーションフレームワーク不使用。Cascade活用
インタラクションVanilla JS (ES5互換)アニメーション・UI制御・フォームIIFE パターン。外部ライブラリ不使用
